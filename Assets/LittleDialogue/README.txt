Little Dialogue is an AddOn for LittleGraph

If you want the script to compile, first you must have LittleGraph package in your project. 
Then you need to add the following scripting symbol in the project settings : "LITTLE_GRAPH"
Then you need to reference certain assemblies:
	- LittleDialogue.Runtime needs to reference LittleGraph.Runtime
	- LittleDialogue.Editor neeeds to reference LittleGraph.Editor & LittleGraph.Runtime
